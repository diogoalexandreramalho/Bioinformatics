Instructions for running in a Mac or Linux OS the Local Alignment algorithm with Python:


1) Open the terminal in the folder where global_alignment.py is.

2) Write in the terminal:
	 >> python3 global_alignment.py

3) Answer the questions that are given to you, for example:

(base) Susana-MacBook-Pro:Python susanavinga$ python3 global_alignment.py
	Introduza a 1ª sequencia: GGCCG
	Introduza a 2ª sequencia: GGATCC
	Insira o match: 3
	Insira o mismatch: -2
	Insira o gap: -4

4) Close the terminal

Instructions for running in a Mac or Linux OS the MSA algorithm with Python:


1) Open the terminal in the folder where Test.py is.

2) Write in the terminal:
	 >> python3 Test.py

3) Answer the questions that are given to you, for example:

(base) Susanas-MacBook-Pro:Python susanavinga$ python3 Test.py 
	Se pretender o algoritmo NW_general pressione 1
	Se pretender MSA pressione 2
	Selecione: 2
	Introduza sequencias na forma AACGTC,AGCGCC,CCCGT,ACAT: AACGTC,AGCGCC,CCCGT,ACAT
	Introduza match: 2
	Introduza mismatch: -1
	Introduza gap: -3

4) Close the terminal


Instructions for running in a Mac or Linux OS the MSA algorithm with Python:


1) Open the terminal in the folder where Test.py is.

2) Write in the terminal:
	 >> python3 Test.py

3) Answer the questions that are given to you, for example:

(base) Susanas-MacBook-Pro:Python susanavinga$ python3 Test.py 
	Se pretender o algoritmo NW_general pressione 1
	Se pretender MSA pressione 2
	Selecione: 1
 	Introduza sequencia na forma GGATCC: ATCG
	Introduza sequencia na forma GGCCG: GTCCA,AGATC
	Introduza match: 2
	Introduza mismatch: -1
	Introduza gap: -3

Note: For the second sequence it is represented an alignment between those sequences in the shape seq1,seq2 without spaces

4) Close the terminal
